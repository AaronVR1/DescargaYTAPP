# DescargaYTAPP
# Es una apliacion desarrollada para facilitar la descarga de videos de Youtube desde nuestro telefono, con unos simples pasos.

# Como descargar y como ejecutarlo:
# 1- Descargamos los archivos de github
# 2- Buscamos donde localizamos la carpeta descargada 
# 3- Abrimos el simbolo del sistema y ejecutamos el archivo (anteriormente se tiene que descargar Yarn y Expo)
# 4- Una vez lo ejecutemos nos generara un codigo QR que tendremos que escanear con la app de Expo GO
# 5- Esperamos a que descargue los archivos necesarios y se ejecutara la APP

# Como usarlo:
# 1- Una vez haya cargado nos mostrara el inicio de la APP con su logo y despues nos aparecera una casilla con botones a los lados
# 2- Dentro de la casilla ingresaremos la URL del video que querramos descargar
# 3- Elegimos la calidad y el formato que queramos descargar
# 4- listo, el video se descargo
